✅ Tu Coaching con IA funciona a través de un **proxy seguro**

¿Cómo funciona?
1. Tu pregunta se envía al proxy
2. El proxy la procesa con GROQ IA
3. Recibes la respuesta aquí
4. **Nunca se expone tu API Key** ✓

💡 Ventajas de esta arquitectura

  • ✓ Seguro: API Keys en el servidor
  • ✓ Privado: No se pueden copiar credenciales
  • ✓ Rápido: GROQ IA (más veloz que OpenAI)
  • ✓ Transparente: Todo el código es visible
  • ✓ Económico: Mejor que OpenAI
